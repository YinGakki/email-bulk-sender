邮件批量发送工具

使用步骤

  1. 配置邮箱 - 修改 .env 文件，填写QQ邮箱SMTP授权码
  2. 导入联系人 - 在"联系人"页面导入CSV/Excel文件
  3. 创建模板 - 在"模板"页面创建邮件模板，支持变量如 {{姓名}}
  4. 撰写邮件 - 选择收件人和模板，编辑内容
  5. 发送邮件 - 点击发送，查看进度和结果

模板变量说明

在邮件内容和主题中可以使用以下变量，系统会自动替换为联系人信息：

变量 说明 示例
{{姓名}} 或 {{name}} 联系人姓名 张三
{{公司}} 或 {{company}} 联系人公司 ABC公司
{{邮箱}} 或 {{email}} 联系人邮箱 zhangsan@example.com
{{电话}} 或 {{phone}} 联系人电话 13800138000
{{备注}} 或 {{notes}} 联系人备注 VIP客户
提示：SMTP 邮箱尚未配置，请前往 设置页面 配置邮箱信息，否则无法发送邮件。
